Componentes

O Vue.js suporta a criação de template para reaproveitar partes do código. Abaixo no código estão dois exemplos de como utilizá-los. A definição da tag de tamplate pode ocorrer fora do app, porém, para exibí-lo é necessário estar em uma tag.

Ordem de declarção
Os componentes devem ser declarados antes da instancia de new Vue({}), caso contrário, não funcionarão. 

// estilo inline
Vue.component('nome-tag01', {
    template: '<input type="text">'
})

Para imprimir a tag seria necessário utilizar a tag abaixo dentro de um app:

<nome-tag01></nome-tag01> 
// utilizando a tag template id com o html.
Vue.component('nome-tag02', {
    template: '#conteudo'
})

Conteúdo a ser replicado: 

<template id="conteudo">
    <ul>
        <li>
            Teste
        </li>
    </ul>
</template>
