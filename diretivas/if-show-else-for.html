Diretivas

O Vue.js possui várias diretivas, também é possível criar diretivas personalizadas.
Em Vue.js as diretivas são códigos que são implementáveis dentro da tag html, e começam sempre com o prefixo vs- Ex de diretivas do Vue.js:

  • v-for
  • v-model
  • v-if
  • v-else

Ex:
<div id='diretiva' -v-if='$instrucao'></div>

v-if

Para exibir o título vá no console do navegador e altere o valor de stauts para true:

$ app01.status = true

Show título: {{titulo}}

v-show

Para exibir o título vá no console do navegador e altere o valor de stauts para true:

$ app02.status = true

Show título: {{titulo}}

v-show VS v-if
A diferença entre utilizar v-show e v-if neste cenário explicitamente é que com o v-if ele não renderiza o título quando status == false.
Com o v-show ele escreve o texto na tela, porém utiliza o css para não exibí-la, ou seja, o elemento existe em tela, só não é visível.

v-else

Para exibir o título vá no console do navegador e altere o valor de stauts para true:

$ app03.status = true

{{titulo}} Não há título.

Para utilizar if e else não é necessário que a tag if e else sejam do mesmo tipo.
