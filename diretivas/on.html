Diretiva on

Utilizado para acessar os métodos da Vue.js.

<button v-on:click="metodo">Enviar</button>

Atalho para o v-on:
É possível substituir o v-on por: arroba ( @ ), ex:
<button @click="metodo">Enviar</button>

Diretiva on e o action do formulário:
É possível utilizar a diretiva v-on junto com a ação de submit do formulário. O primeiro evento a ser disparado é o v-on, no caso do exemplo acima, é um click, e posteriormente é executado o submit de fato.

Também é possível cancelar o envio do formulário, só disparar a função definida no v-on.
