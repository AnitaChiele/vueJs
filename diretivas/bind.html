Diretiva bind

Para imprimir dentro de atributos das tags de html é necessário utilizar o bind.

<input type="text" v-bind:value="nome">

Atalho para o v-bind:
É possível substituir o v-bind por: dois pontos ( : ), ex:
<input type="text" :value="nome">
