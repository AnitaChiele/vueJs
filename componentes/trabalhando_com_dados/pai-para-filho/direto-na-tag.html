Passando os dados por componentes direto na tag


É possível passar informações do componente pai para o componente filho direto na tag, ou seja, com os dados chumbados. Utilizaremos as propriedades ( props ) do Vue para enviar os dados para o componente filho.

Quem é quem no exemplo acima? O componente pai é o: nav01 que é o template id: cpt-nav01.
O componente filho é o: menu01 que é o template id: cpt-menu01.


Componente Pai

Vamos chamar o componente filho:

<componente-filho></componente-filho>
Componente Filho

Para ler os dados é necessário imprimir no a variável definida na props do componente filho do VueJS:

<li v-for="i in var-props">{{i.label}}</li>

Ex:
<li v-for="item in itens">{{item.label}}</li>

Definição do componente filho no VueJs: 

Vue.component('cpt-menu01', {
    template: '#menu01',
    props: ['var-props']
})

Ex: 

Vue.component('cpt-menu01', {
    template: '#menu01',
    props: ['itens']
})
