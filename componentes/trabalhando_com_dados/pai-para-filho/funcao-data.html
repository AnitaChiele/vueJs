Passando os dados por componentes pela função data

Quem é quem no exemplo acima? O componente pai é o: nav01 que é o template id: cpt-nav01.
O componente filho é o: menu01 que é o template id: cpt-menu01.

Ainda no fluxo de passar os dados do componente pai para o componente filho é possível passar os dados através da função data em vez de chumbado na tag. Para isso é necessário definir uma variável no componente pai - como já foi mostrado em direto na tag, e setar no js do componente pai os valores.

Componente Pai

No html é necessário chamar o componente filho, passar uma variável e definir uma variável que irá ter seus valores definidos no VueJS do componente pai:

<cpt-filho :var-itens="var-content"></cpt-filho>

[14.02] Ex:
<cpt-menu01 :itens="itens-menu"></cpt-menu01>

No Vue defina o valor de var-content: 

Vue.component('cpt-pai', {
    template: '#qualquer-template',
    data(){
        return{
            var-itens: [
                {label: 'Pagina inicial'},
                {label: 'Sobre'},
                {label: 'Contato'},
            ]
        }
    }
})
Componente Filho

Para ler os dados enviados pelo componente pai, é necessário imprimir a variável no html:

<li v-for="i in var-itens">{{i.label}}</li>

Levando em consideração os nomes definos no ex [14.02]: <li v-for="item in itens">{{item.label}}</li>

E no VueJS é necessário indicar qual é o nome da variável que foi definido para chamar o componente filho no componente pai: 

Vue.component('cpt-filho', {
    template: '#qualquer-template2',
    props: ['var-itens']
})

Levando em consideração os nomes definos no ex [14.02]: 

Vue.component('cpt-menu01', {
    template: '#menu01',
    props: ['itens']
})
