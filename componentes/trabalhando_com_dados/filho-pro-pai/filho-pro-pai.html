Enviando dados do componente filho pro componente pai


Para fazer transferência de dados a partir de um componente filho para o componente pai é necessário utilizarmos o $emit. Ele permitirá a criação de eventos personalizados, além dos eventos padrões (onClick, onFocus, e etc). Para 'escutar' a resposta do $emit basta utilizar o mesmo evento no componente desejado, nesse caso, no componente pai.

Quem é quem no exemplo acima? O componente filho é o: menu05.
O componente pai é o: nav05.


Componente Filho

Para o enviar as informações para o componente pai é necessário criar um evento personalizado, na tag html deve ter:

@click="nomeEventoCriado(Param)"

No vue: 

methods: {
    nomeEventoCriado (param) {
        this.$emit('nome-do-evento', index)
    }
}
Componente Pai

Para pegar os dados enviados do compoente filho é necessário criar uma função que lide com essas informações no componente pai, na tag html deve ter:

@nome-do-evento="nomeFuncaoQueVaiLer"

No vue: 


methods: {
    nomeFuncaoQueVaiLer(param){
        alert("os dados chegaram no pai!")
    }
}
