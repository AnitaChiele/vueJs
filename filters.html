Filters

No Vue.js é possível criar filtros, para isso existe um método chamado: filters. O primeiro parâmetro é sempre a própria string. Utilizamos o pipe ( | ) para a chamada do filter. 

new Vue({
    el: "#app01",
    data:{},
    filters:{
        nomeFuncao(param1){}
    }
})
